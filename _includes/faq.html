FAQ: Frequently Asked Questions

Alcune informazioni che possono esserti utili.

Cos'è GitHub?

GitHub è un servizio web di hosting per lo sviluppo di progetti software, che usa il sistema di controllo di versione Git.

Vedi Wikipedia

Esistono alternative a GitHub per il mio sito?

Certo, esistono alternative gratuite (Vedi Netsons & Hostinger), di solito questi tipi di servizi offrono anche possibilità più avanzate rispetto a GitHub, come php e MySql, che però se per un lato favoriscono la creazione di un sito più complesso e personalizzabile, da un lato sono meno user-friendly per nuovi utenti del web.

Cos'è un Repository?

Il repository è un archivio ordinato dove sono raccolti i file di uno specifico progetto, in modo ben organizzato e costantemente aggiornato.

Vedi Wikipedia
